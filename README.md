 Online Hackathon Management System
 
 Project Overview

The Online Hackathon Management System is a web application designed to manage hackathons efficiently. It provides a platform where organizers can create and manage hackathons, participants can register and submit projects, and judges can evaluate submissions.

This system simplifies the entire hackathon workflow including registration, team formation, project submission, evaluation, and result announcement.

 Objectives

* To simplify hackathon event management.
* To provide a platform for participants to collaborate and submit projects.
* To allow judges to evaluate projects easily.
* To display leaderboards and winners transparently.

 User Roles

 Participants

Participants can:

* Register and login
* Join or create teams
* View available hackathons
* Submit project details
* View results and leaderboard

 Organizer / Admin

Organizers can:

* Create and manage hackathons
* Manage participants and teams
* Assign judges
* Monitor project submissions
* Publish final results
